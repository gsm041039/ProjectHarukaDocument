210_角色視覺設計檔案_小光 (Rev. 1.0)

## **1. 角色基礎資訊**

**全名：** 小光 (Ko Hikaru)

**年齡：** 8-10歲（死亡時）

**身份：** 夜區貧困小學生 → 魔法屍骸（Type B 廢鐵型）

**角色定位：** 朱音的精神支柱、悲劇的觸發點、「純真被摧毀」的象徵

**核心關鍵詞：** 純真 (Innocence)、飢餓 (Hunger)、生鏽的希望 (Rusty Hope)、悲憐 (Pity)

**象徵意義：**
- 朱音內心最後的溫暖錨點
- 代表「還有值得保護的純真」
- 死亡 = 朱音從「保護人類」轉向「保護屍骸」的轉折點
- 屍骸化 = 「愛的對象變成怪物，但依然值得愛嗎？」的道德拷問

---

## **2. 生前形態 (Human Form) - 「夜區的螢火蟲」**

**設計核心：** 貧困但溫暖、弱小但堅強、微光般的存在

### **2.1 外觀設計**

**體型：**
- 身高：約130-140cm（偏瘦小，營養不良）
- 體型：瘦弱，肋骨隱約可見，但不是病態的瘦
- 姿態：略微駝背（習慣性防禦姿勢），但眼神明亮

**髮型：**
- 款式：凌亂的短髮，像很久沒剪過
- 顏色：深褐色，但因為營養不良而有些乾枯、無光澤
- 細節：瀏海遮住一隻眼睛（內向、害羞的視覺暗示）
- 特徵：後腦勺有幾根翹起的呆毛（唯一的「可愛」元素）

**臉部特徵：**
- 眼睛：大而明亮（對比瘦弱的身體），深褐色瞳孔
- 表情：靦腆但溫暖的笑容，露出缺了一顆的門牙（換牙期）
- 皮膚：略帶灰色調（長期營養不良），臉頰有些凹陷
- 特徵：眼下有淡淡的黑眼圈（睡眠不足）

**服裝：**
- **上身**：過大的二手T恤
  - 顏色：褪色的藍色或灰色
  - 細節：領口鬆垮、袖子過長（需要捲起來）
  - 污漬：袖口和領口有洗不掉的污漬
- **下身**：補丁褲子
  - 款式：短褲或七分褲（因為長褲太貴）
  - 細節：膝蓋處有明顯的補丁（不同顏色的布料）
  - 磨損：褲腳邊緣磨損嚴重
- **鞋子**：破舊的布鞋或涼鞋
  - 狀態：鞋底磨平、鞋帶打結（因為斷過）
  - 細節：左右腳可能不完全一樣（拾荒或救濟品）

**標誌物：小學生書包**
- **外觀**：紅色的小學生書包（唯一「正常」的物品）
- **狀態**：
  - 顏色褪色成暗紅色
  - 肩帶磨損，有縫補痕跡
  - 書包上貼著舊的卡通貼紙（已經褪色、剝落）
- **內容物**：
  - 幾本皺巴巴的課本
  - 用到很短的鉛筆（捨不得丟）
  - 橡皮擦（已經用到很小塊）
  - 一個空的飯盒（經常吃不飽）

**關鍵道具：波板糖（彩色螺旋棒棒糖）**
- **外觀**：彩色螺旋狀的棒棒糖（彩虹色：紅、橙、黃、綠、藍）
- **象徵意義**：
  - 小光僅有的「奢侈品」
  - 他願意分享給朱音 = 純粹的善意
  - 生前最後給朱音的糖 = 死後朱音的執念
- **視覺處理**：
  - 生前：鮮豔、透明、甜美
  - 死後（遺物）：發霉發黑、黏著灰塵

### **2.2 動作與行為設計**

**行走方式：**
- 步伐：小碎步，略微拖著腳（體力不足）
- 姿態：肩膀向內縮（習慣性防禦姿勢）
- 書包：因為太重，身體會向一側傾斜

**與朱音互動時：**
- **眼神**：從下往上看（身高差），眼神中有仰慕和依賴
- **動作**：
  - 遞糖果時：小心翼翼地從口袋掏出，雙手捧著遞給朱音
  - 微笑時：露出缺牙的笑容，眼睛瞇成月牙
  - 害羞時：用手指搓著衣角，腳尖在地上畫圈
- **聲音**：
  - 音調：童聲，略帶沙啞（營養不良影響發育）
  - 說話方式：輕聲細語，有些結巴
  - 關鍵台詞：「家姐……呢粒糖……俾你食……」

**平時狀態：**
- 經常餓著肚子，會不自覺地摸肚子
- 看到食物時眼睛會發光，但不會主動要
- 在廢墟中翻找可用的東西（拾荒兒童的習慣）

### **2.3 色彩指引（生前）**

| **色系**           | **具體顏色**             | **象徵意義**           |
| ------------------ | ------------------------ | ---------------------- |
| **主色（Base）**   | 褪色的藍灰色、暗紅色     | 貧困、夜區的灰暗       |
| **對比色（Accent）** | 彩色波板糖的鮮豔色       | 微小的希望與快樂       |
| **皮膚色調**       | 灰白色、營養不良的蒼白   | 生存困境               |
| **光效**           | 柔和的逆光（像螢火蟲）   | 微弱但溫暖的存在       |

**氛圍關鍵詞：**
- **螢火蟲 (Firefly)**：微弱但溫暖的光
- **廢墟中的花 (Flower in Ruins)**：在絕望中依然保持純真
- **最後的糖果 (The Last Candy)**：唯一能給予的善意

## **3. 屍骸形態 (Corpse Form) - 「生鏽的孩子」**

**設計核心：** 可憐的畸形兒，而非恐怖怪獸；讓人心痛而非恐懼

**屍骸類型：** Type B 廢鐵型（Scrap Metal Type）

**設計理念：**
- **不是「怪物」**：保留兒童的大致輪廓，讓人能認出「這曾經是個孩子」
- **不是「恐怖」**：動作是「渴求」而非「攻擊」，像飢餓的小動物
- **情感目標**：玩家/觀眾看到時應該感到**心痛**而非恐懼

### **3.1 外觀設計**

**體型：**
- 高度：約100-120cm（比生前更矮小，身體扭曲）
- 體型：扭曲但保留兒童的比例（大頭、短手腳）
- 姿態：佝僂、顫抖，像隨時會倒下

**材質與構造：**
- **主體**：廢鐵、生鏽的金屬碎片拼湊而成
- **金屬種類**：
  - 鐵皮（生鏽、變形）
  - 螺絲釘、鐵釘（突出體表）
  - 金屬絲（像血管一樣纏繞）
- **接縫**：
  - 金屬片之間用粗糙的焊接或鐵絲綁定
  - 接縫處滲出黑色的焦油狀液體（屍骸體液）
- **質感**：
  - 生鏽：橙褐色、鐵紅色的鏽跡
  - 磨損：金屬表面有刮痕、凹陷
  - 聲音：移動時發出「咯吱咯吱」的金屬摩擦聲

**頭部設計：**
- **形狀**：保留兒童的大致輪廓，但扭曲變形
- **「臉」**：
  - 沒有明確的五官，只有凹陷的「眼窩」和「嘴巴」
  - 眼窩：空洞，但深處有微弱的紅光（像煤炭餘燼）
  - 嘴巴：裂開的金屬縫隙，內部漆黑
- **頭髮**：
  - 用生鏽的鐵絲模擬頭髮，凌亂地從頭部伸出
  - 有些鐵絲斷裂，垂在「臉」前

**四肢設計：**
- **手臂**：
  - 細長而扭曲，由金屬管和碎片組成
  - 手掌：鉗子狀或鉤子狀，但動作是「伸手」而非「抓取」
  - 動作：顫抖著向前伸，像渴求擁抱
- **腿部**：
  - 不對稱（一長一短），行走不穩
  - 關節處發出「咯噠咯噠」的聲音
  - 移動方式：拖著腳走，偶爾會絆倒自己

**背部設計：**
- **書包**：依然掛著那個紅色的小學生書包
  - 狀態：破爛不堪，被金屬碎片刺穿
  - 肩帶：纏繞在金屬身軀上，像鐐銬
  - 內容物：課本和鉛筆從破洞中掉出，散落地上
  - **視覺衝擊**：「這是個孩子」的唯一證據

**關鍵道具：波板糖（屍骸版本）**
- **位置**：依然握在「手」中，或插在身體某處
- **狀態**：
  - 發霉發黑，表面覆蓋黑色黴菌
  - 彩色螺旋已經變成灰黑色
  - 糖體融化變形，黏著鐵鏽和灰塵
  - 但依然能辨認出是「波板糖」
- **象徵意義**：
  - 死前最後的執念
  - 想要分享給朱音的「最後的糖果」
  - 成為朱音屍骸化後的聖物

### **3.2 動作與行為設計**

**核心動作邏輯：渴求而非攻擊**

**行走方式：**
- 蹣跚、不穩，像剛學走路的嬰兒
- 經常絆倒，但會掙扎著站起來
- 發出「咯吱咯吱」的金屬聲和「咯噠咯噠」的腳步聲

**與朱音互動：**
- **被抱住時**：
  - 身體僵硬，但不反抗
  - 頭部會輕微傾斜，像是依偎
  - 眼窩中的紅光變得柔和
- **被餵食時**（朱音餵情緒結晶）：
  - 張開「嘴」（金屬裂縫），顫抖地接受
  - 發出低頻的「嗡嗡」聲（滿足的聲音？）
  - 餵食後會安靜下來，像睡著一樣

**失控狀態（最終場景）：**
- **觸發點**：飢餓到極限，本能壓倒意識
- **動作變化**：
  - 從「渴求」變成「瘋狂抓取」
  - 手臂揮舞，金屬碎片割傷朱音
  - 眼窩中的紅光變得刺眼、閃爍
  - 發出刺耳的金屬尖嘯聲
- **朱音的反應**：
  - 即使被攻擊，依然抱著他
  - 台詞：「唔緊要……家姐唔怕痛……乖，唔好驚……」

### **3.3 聲音設計**

**設計原則：不是恐怖的咆哮，而是心痛的哭聲**

**平時聲音：**
- **金屬摩擦聲**：「咯吱咯吱」（移動時）
- **低頻嗡鳴**：像老舊機器的運轉聲，但音調悲傷
- **失真的童聲**：
  - 經過電子失真處理的「肚餓……」（廣東話：餓了）
  - 「家姐……」（朱音）
  - 聲音斷斷續續，像壞掉的錄音機

**情緒變化：**
- **平靜時**：低頻嗡鳴 + 規律的金屬聲
- **渴求時**：童聲「肚餓……肚餓……」+ 顫抖的金屬聲
- **失控時**：刺耳的尖嘯 + 混亂的金屬碰撞聲 + 失真的哭聲

**聽覺目標：**
- 第一次聽到：「這是什麼怪物的聲音？」
- 辨認出是小光後：「天啊……他在哭……」
- 最終處決時：「對不起……對不起……」（玩家/觀眾的心聲）

### **3.4 色彩指引（屍骸形態）**

| **色系**           | **具體顏色**                     | **象徵意義**           |
| ------------------ | -------------------------------- | ---------------------- |
| **主色（Base）**   | 生鏽的鐵灰色、橙褐色             | 腐朽、廢棄             |
| **對比色（Accent）** | 暗紅色（書包）、黑色（焦油液體） | 僅存的身份標記、絕望   |
| **點綴（Detail）**  | 發霉的黑色（波板糖）             | 扭曲的善意             |
| **光效（Lighting）** | 眼窩的紅色微光                   | 微弱的「生命」         |

**氛圍關鍵詞：**
- **生鏽的孩子 (Rusty Child)**：曾經的純真被鏽蝕
- **飢餓的機器 (Hungry Machine)**：不是怪物，是餓壞的孩子
- **破碎的書包 (Broken Schoolbag)**：身份的最後證明
- **發霉的糖果 (Moldy Candy)**：變質的善意

## **4. 情感與象徵分析**

### **4.1 生前的小光：溫暖的錨點**

**對朱音的意義：**
- **唯一的善意**：在充滿惡意的世界裡，小光是唯一對她好的人
- **生存的理由**：「如果我死了，就沒人保護小光了」
- **人性的證明**：「我還值得被愛」的證據

**視覺象徵：**
- **波板糖**：小光的善意，朱音的希望
- **書包**：「他是個普通的孩子，有資格活下去」
- **缺牙的笑容**：純真、無邪、不設防

### **4.2 死後的小光：道德困境的化身**

**三位一體問題的核心：**
- **問題一**：「小光變成屍骸後，還是小光嗎？」
- **問題二**：「如果他依然是小光，殺他是謀殺；如果他不是，朱音在保護什麼？」
- **問題三**：「當愛的對象變成怪物，愛是否應該繼續？」

**視覺象徵：**
- **生鏽的身軀 vs 破爛的書包**：「怪物」與「孩子」的雙重身份
- **發霉的波板糖 vs 伸出的手**：「扭曲的善意」依然是善意嗎？
- **失控的攻擊 vs 朱音的擁抱**：「無法被拯救」但「不願放棄」

### **4.3 最終處決：悲劇的高潮**

**場景重現：**
1. 小光失控，攻擊朱音
2. 朱音依然抱著他：「乖，唔好驚，家姐喺度……」
3. 晴香舉起武器，卻無法下手——朱音的眼神裡有太多她認識的東西
4. 小光在失控中能量耗盡，金屬身軀碎裂，紅光熄滅，在朱音懷中消散
5. 朱音抱著空氣，發出無聲的嘶吼
6. 晴香：「……滾。唔好再俾我見到妳。」
7. 朱音踉蹌離去，消失在下水道入口
8. 掉落：發霉的波板糖（小光的遺物）

**玩家/觀眾應有的情緒：**
- **對峙時**：「我不想做這個選擇……」
- **小光消散時**：「他終於……不用再痛苦了……」
- **放逐時**：「這樣對嗎？」「不殺她，是軟弱還是正確？」

**長期影響：**
- 玩家在之後看到小孩屍骸時，會想起小光
- 對「消滅屍骸 = 正義」的信念產生動搖
- 朱音的放逐為後續「嘆息之橋戰役」的回歸埋下伏筆
- 為後續的「三位一體光譜」哲學討論埋下種子

## **5. 演出指導與分鏡參考**

### **5.1 初次登場（生前）**

**場景：** 夜區廢墟，戰鬥後的黃昏

**分鏡：**
1. 遠景：朱音坐在廢墟上，滿身是血
2. 腳步聲（咯噠咯噠）
3. 中景：小光拖著書包走過來
4. 特寫：小光從口袋掏出波板糖，雙手捧著
5. 小光：「家姐……呢粒糖……俾你食……」
6. 朱音愣住，特寫她顫抖的手接過糖果
7. 小光露出缺牙的笑容，眼睛瞇成月牙
8. 朱音的眼淚滴在糖果上

**視覺重點：**
- 逆光處理：小光像微弱的光
- 色彩對比：小光的彩色波板糖 vs 朱音的黑紅血污
- 音樂：溫柔的鋼琴旋律（後續成為小光的主題曲）

### **5.2 死亡場景（屍骸化轉變）**

**場景：** 夜區大規模屍骸襲擊，廢墟戰場

**分鏡：**
1. 遠景：朱音在戰場上揮舞電鋸，屍骸四散
2. 朱音突然停下（戒斷反應），電鋸掉落
3. 朱音視角：畫面模糊、顫抖，耳鳴聲（表現戒斷狀態）
4. 遠處，小光躲在廢墟後，抱著書包顫抖
5. 一隻屍骸發現小光，緩慢接近
6. 朱音想動，但身體不聽使喚：「唔好……小光……走啊……」
7. 特寫：小光轉頭看向朱音，眼中是恐懼和求救
8. 屍骸撲向小光
9. 朱音的尖叫：「唔好啊啊啊啊！！！」
10. 黑屏，只有小光的慘叫聲
11. 慢慢淡入：小光的屍體，周圍是金屬碎片和血
12. 朱音爬過去，抱起小光：「對唔住……對唔住……」
13. 小光的身體開始變化：金屬碎片從皮膚下長出，眼睛變成空洞
14. 朱音崩潰：「唔好……唔好走……」
15. 黑奏的聲音（畫外音）：「他沒有死，只是換了生存方式。」

**視覺重點：**
- 色彩：從彩色迅速轉為灰黑（朱音的世界崩塌）
- 音效：戒斷時的耳鳴 + 小光的慘叫 + 金屬變化聲
- 剪輯：快速剪輯（混亂）→ 慢鏡頭（悲劇時刻）→ 靜止（絕望）

### **5.3 最終處決場景**

**場景：** 屍骸樂園（廢棄糖果工廠），糖果王座

**分鏡：**
1. 晴香團隊進入工廠，看到朱音抱著小光坐在王座上
2. 朱音溫柔地搖晃小光：「乖，唔餓喇……家姐餵飽你喇……」
3. 小光的金屬身軀發出低頻嗡鳴（平靜）
4. 晴香：「朱音……放低佢啦……」
5. 朱音抬頭，破碎的光環閃爍：「點解？佢係我嘅孩子。」
6. 操：「佢已經唔係小光啦……」
7. 朱音：「點解人類有生存權，屍骸就要被殺？佢哋都係生命！」
8. 三位一體辯論（對話場景）
9. 突然，小光掙扎，眼窩紅光閃爍，發出尖嘯
10. 朱音緊抱他：「唔好驚，家姐喺度……」
11. 小光失控，鋒利的金屬割傷朱音
12. 朱音的血（黑紅色 + 糖漿質感）滴在小光身上
13. 晴香舉起武器，但手在顫抖——朱音的眼神讓她無法下手
14. 小光在失控中能量耗盡，金屬身軀開始崩解
15. 糖晶硬殼碎裂，光環熄滅，小光在朱音懷中消散（慢鏡頭）
16. 朱音抱著空氣，發出無聲的嘶吼
17. 晴香：「……滾。唔好再俾我見到妳。」（聲音顫抖）
18. 朱音踉蹌站起，處於半屍骸半人類狀態，消失在黑暗中
19. 地上：發霉的波板糖、破爛的書包、破碎的光環碎片
20. 晴香跪下，撿起波板糖，無聲地哭泣

**視覺重點：**
- 構圖：朱音和小光始終在畫面中心（母子像）
- 光影：破碎光環的閃爍 + 小光眼窩的紅光 = 不穩定的美
- 色彩：腐爛的糖果色 + 黑紅血液 = 甜美與絕望的混合
- 音樂：小光的主題曲（鋼琴）變成破碎、失真的版本

## **6. 與其他角色的視覺關聯**

### **6.1 小光 vs 朱音（屍骸女王）**

| **視覺元素** | **小光（生前）**       | **小光（屍骸）**       | **朱音（屍骸女王）**   |
| ------------ | ---------------------- | ---------------------- | ---------------------- |
| **色彩**     | 彩色波板糖（希望）     | 發霉的黑色（扭曲）     | 腐爛的糖果色（絕望）   |
| **材質**     | 柔軟的布料（人性）     | 生鏽的金屬（物化）     | 糖晶硬殼（半屍骸化）   |
| **光效**     | 逆光（螢火蟲）         | 眼窩紅光（餘燼）       | 破碎光環（瀕臨崩潰）   |
| **象徵**     | 純真的錨點             | 扭曲的善意             | 瘋狂的母性             |

**共同元素：**
- **書包**：小光依然背著，朱音在王座旁堆滿小孩文具
- **波板糖**：小光握著發霉的糖，朱音的調味粉罐
- **母子構圖**：朱音永遠抱著小光，像聖母與聖子的扭曲版

### **6.2 小光 vs 晴香（對比）**

| **視覺元素** | **小光**                   | **晴香**                   |
| ------------ | -------------------------- | -------------------------- |
| **角色定位** | 需要被保護的弱者           | 保護他人的英雄             |
| **服裝**     | 破爛的二手衣               | 華麗的粉紅魔法少女服       |
| **象徵**     | 微弱的希望（螢火蟲）       | 耀眼的希望（太陽）         |
| **結局**     | 被殺死 → 變成屍骸 → 被處決 | 活下來但背負罪惡感         |

**對比意義：**
- 小光的死證明了「不是所有人都能被拯救」
- 晴香的罪惡感源於「我沒能保護小光，最後還親手殺了他」

## **7. 製作參考與靈感來源**

**視覺參考：**
- **生前小光**：宮崎駿《螢火蟲之墓》（貧困兒童的真實感）
- **屍骸小光**：《寂靜嶺》的扭曲兒童怪物（但更悲傷而非恐怖）
- **母子構圖**：米開朗基羅《聖母憐子圖》（悲劇的神聖感）
- **色彩**：Tim Burton《屍新娘》（腐爛但依然美麗）

**情感參考：**
- 《進擊的巨人》：親人變成巨人後的兩難
- 《最後生還者》：Ellie 無法被治癒的感染者
- 《魔法少女小圓》：沙耶香的墮落（善意導致毀滅）

**聲音參考：**
- 失真的童聲：《寂靜嶺》的廣播噪音 + 兒童笑聲
- 金屬摩擦聲：《異形》的機械音效
- 背景音樂：久石讓的鋼琴曲（溫柔但悲傷）

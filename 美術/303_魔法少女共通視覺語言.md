## 魔法少女共通視覺語言 (Magical Girl Common Visual Language)

### 引言：設計哲學

---

本文件定義了所有魔法少女共享的視覺符號系統。不同於魔法屍骸的「齒輪+鏡子」符號（見 [302_魔法屍骸視覺設計檔案](302_魔法屍骸視覺設計檔案.md)），魔法少女的視覺語言強調：

- **「仍然活著」**：以非物質的能量體對比屍骸的實體金屬
- **「代價的隱晦」**：枷鎖以紋理形式存在，而非外露的機械
- **「鏡像的演變」**：變身過程中的鏡子，從舞台變成監牢

**核心對比原則：**

| 魔法少女 | 魔法屍骸 |
|---------|---------|
| 能量體（流動、發光） | 實體金屬（停滯、生鏽） |
| 隱晦的代價 | 外露的侵蝕 |
| 與鏡中倒影互動 | 只能在碎鏡中映照扭曲 |

---

### 第一部分：核心視覺符號系統

#### 1.1 光環 (Halo) - 力量的榮耀

**定義：** 每位魔法少女在魔法形態下都擁有的、非物質的高能量體。這是她們與【集體潛意識】連結的視覺證明，也是「仍然活著」的象徵。

##### 1.1.1 核心特性

- **材質**：非物質的純能量體，不可觸碰
- **光學特性**：自發光，帶有微弱的粒子飄散效果
- **動態**：隨著心匣的「心跳」輕微脈動

##### 1.1.2 共通性與個人化

| 屬性 | 共通性 | 個人化 |
|-----|-------|--------|
| 存在 | 所有魔法少女都有光環 | - |
| 形狀 | - | 因人而異（圓形、心形、星形等） |
| 位置 | - | 因人而異（頭頂、背後、心口等） |
| 顏色 | - | 對應個人主題色 |

##### 1.1.3 狀態變化

| 狀態 | 光環表現 |
|-----|---------|
| **健康** | 穩定發光，輪廓清晰，有規律的脈動 |
| **疲勞** | 光芒減弱，偶爾閃爍 |
| **損耗嚴重** | 出現裂紋/缺口，光芒不穩定 |
| **瀕臨崩潰** | 劇烈閃爍，開始碎裂 |

##### 1.1.4 與魔法屍骸的對應

魔法少女的「光環」在墮落為屍骸後，會轉化為「光環齒輪」(Halo Gear)：

| 魔法少女 | 魔法屍骸 |
|---------|---------|
| 非物質能量體 | 實體金屬齒輪 |
| 自由漂浮 | 固定在後腦 |
| 發光、脈動 | 停止轉動或強制運轉 |
| 代表「仍然連結著」 | 代表「被困在系統中」 |

---

#### 1.2 枷鎖紋 (Shackle Mark) - 契約的代價

**定義：** 出現在魔法少女身體上的圈狀紋理，代表與力量簽訂契約所付出的代價。這是「束縛」的隱晦象徵，對比屍骸身上外露的齒輪軌道。

##### 1.2.1 核心特性

- **形態**：圈狀紋理，環繞身體特定部位
- **可見度**：可被衣服遮蓋，細看才會發現
- **材質感**：如同燙印在皮膚上的印記，帶有微弱的光澤

##### 1.2.2 共通性與個人化

| 屬性 | 共通性 | 個人化 |
|-----|-------|--------|
| 存在 | 所有魔法少女都有枷鎖紋 | - |
| 形狀 | 圈狀紋理 | 紋理細節因人而異 |
| 位置 | - | 因人而異（手腕、腳踝、頸部、腰部等） |
| 數量 | - | 初始位置數量因人而異 |

##### 1.2.3 演變規則：範圍擴散

枷鎖紋會隨著靈魂損耗而**範圍擴散**：

| 階段 | 枷鎖紋狀態 |
|-----|-----------|
| **初期** | 只在一兩個位置有細微的圈紋 |
| **中期** | 紋理變深，開始向周圍擴散 |
| **後期** | 多處出現枷鎖紋，紋理連成網狀 |
| **瀕臨崩潰** | 枷鎖紋遍布全身，開始外露如傷口 |

##### 1.2.4 與魔法屍骸的對應

| 魔法少女 | 魔法屍骸 |
|---------|---------|
| 枷鎖紋（隱晦紋理） | 齒輪軌道（外露金屬） |
| 印在皮膚上 | 刺穿肉體、撕裂肌肉 |
| 可被衣服遮蓋 | 無法隱藏 |
| 代表「接受的代價」 | 代表「被強制的同化」 |

---

### 第二部分：變身序列 - 鏡子三階段系統

所有魔法少女的變身都涉及**鏡子**，呼應「鏡中的承諾」契約儀式（見 [魔法少女設定詳解](../編劇/魔法少女設定詳解：心匣、裝置與宿命.md)）。

隨著角色的精神狀態變化，變身過程中的鏡子會經歷三個階段的演變：

---

#### 2.1 階段一：夢幻之鏡 (The Idol / The Dream)

**關鍵詞：** 華麗、自願、舞台感

**適用時期：** 角色仍然相信自己是正義的英雄，享受著力量帶來的榮耀。

##### 鏡子的表現

- **外觀**：如同化妝間的明亮鏡台，或通往舞台的大門
- **鏡框**：裝飾華麗，帶有蕾絲、寶石光效
- **光線**：溫暖、柔和、帶有夢幻的光暈

##### 倒影的互動

- 鏡中的倒影是**完美、自信、閃閃發光**的自己
- 倒影會主動向少女**伸出手邀請**
- 少女是**主動、開心**地握住倒影的手
- 可能會有**跳舞旋轉穿過鏡面**的優雅動作

##### 著裝過程

- 鏡面化作**無數閃粉和花瓣**
- 服裝如同被光芒編織而成
- 可能有**羽毛飄散**的夢幻特效

##### 枷鎖的表現

- 枷鎖在這個階段看起來像**「首飾」**（手鐲、頸鏈）
- **完全沒有束縛感**，只是美麗的裝飾
- 出現時伴隨著**清脆悅耳的鈴聲**

##### 氛圍與玩家感受

> 「好靚！我要拯救世界！」

---

#### 2.2 階段二：裂痕之鏡 (The Doubt / The Crack)

**關鍵詞：** 違和感、猶豫、機械音

**適用時期：** 角色開始發現真相，戰鬥帶來的精神壓力開始浮現。

##### 鏡子的表現

- 鏡面偶爾會閃過**雜訊 (Glitch)**
- 鏡框開始出現**細微的裂紋**
- 變身背景音樂可能會有一兩秒**變調**，或混入**齒輪轉動的聲音**

##### 倒影的互動

- 倒影會**遲疑一秒**才伸出手
- 倒影的**表情比本體更加冷漠**
- 少女穿過鏡面時，會有**少少阻力**，如同行入深水

##### 著裝過程

- 鏡片破碎的聲音變得**清脆刺耳**（玻璃聲）
- 服裝出現的過程不再完全流暢
- 可能會有短暫的**畫面凍結**

##### 枷鎖的表現

- 枷鎖出現時，會有輕微的**「咔嚓」上鎖聲**
- 枷鎖的光澤開始變得**略微暗淡**
- 細看會發現枷鎖比之前**更緊**了一點

##### 氛圍與玩家感受

> 「咦？點解個變身好似怪怪地？個笑容有啲僵硬？」

---

#### 2.3 階段三：監禁之鏡 (The Puppet / The Cage)

**關鍵詞：** 恐怖、強制、詭異微笑

**適用時期：** 心靈已經被體制同化，或者處於崩潰邊緣。

##### 鏡子的表現

- 鏡子變成**四面圍堵的監牢**
- 不再是一面鏡子，而是**無處可逃的鏡之牢籠**
- 光線變得**冰冷、高對比度 (Hard Light)**

##### 倒影的互動

- 倒影**面無表情**或帶著**詭異的微笑**
- 倒影不再是邀請，而是**強制抓捕**
- 少女可能**面露恐懼**，被鏡中的倒影**強行拉入**融合
- 或者少女已經**完全順從**，面無表情地走入鏡中

##### 著裝過程

- 鏡片碎片**刺入身體**，變成盔甲
- 枷鎖的聲音變成**沉重的鐵鍊聲**
- 整個過程帶有**機械的精確感**，失去了人性的溫度

##### 枷鎖的表現

- 枷鎖不再是首飾，而是**真正的鐵鏈**
- 可以清晰看到枷鎖**深深陷入皮膚**
- 枷鎖的材質開始**接近金屬**（預示著向屍骸的轉變）

##### 最終定格

- 少女擺出**華麗的Pose**
- 但眼神是**死寂的**
- 嘴角可能帶著**完美但僵硬的微笑**

##### 氛圍與玩家感受

> 「......」

### 第三部分：心匣共通視覺語法

心匣的外觀保持各角色的獨特性，但在某些核心表現上遵循統一的視覺語法。

#### 3.1 心跳聲

**共通性：** 所有心匣在打開時都會發出「心跳聲」。

**個人化：** 每個角色的心跳節奏不同，對應其性格：

| 角色 | 心跳節奏參考 |
|-----|-------------|
| 晴香 | 活潑、略快，偶爾不規則（焦慮） |
| 美夜子 | 緩慢、沉穩，如同冰下的心跳 |
| 凜 | 精確、規律，如同節拍器 |

#### 3.2 損壞表現

**統一語法：** 所有心匣的損壞都呈現「瓷器龜裂」的紋理。

| 損壞程度 | 視覺表現 |
|---------|---------|
| **輕微** | 表面出現一兩道細微裂紋 |
| **中度** | 裂紋增多，開始滲出微弱的光 |
| **嚴重** | 裂紋遍布，有碎片開始剝落 |
| **瀕臨崩潰** | 只靠微弱的光芒勉強維持形態 |

#### 3.3 自然覺醒者的特殊表現：金繼

**適用對象：** 只限於透過「鏡中的承諾」自然覺醒的魔法少女（未使用情緒力量裝置）。

**視覺特徵：** 心匣上的裂紋會被**金色/發光的物質**填補，如同日本「金繼」工藝。

**象徵意義：** 「破碎但用光修補」——傷痕成為美麗的一部分，而非純粹的損傷。

### 第四部分：與魔法屍骸的鏡像對應

本節整理魔法少女視覺元素與魔法屍骸的對應關係，作為設計時的參考。

#### 4.1 總覽對照表

| 元素 | 魔法少女 | 魔法屍骸 |
|-----|---------|---------|
| **核心符號** | 光環（能量體） | 光環齒輪（金屬） |
| **束縛象徵** | 枷鎖紋（隱晦紋理） | 齒輪軌道（外露侵蝕） |
| **鏡像元素** | 變身鏡（完整、可互動） | 碎鏡片（只映照扭曲） |
| **能量核心** | 心匣（有心跳、發光） | 核心（死寂或過載） |
| **材質感** | 有機、溫暖、發光 | 無機、冰冷、生鏽 |
| **動態** | 流動、脈動、呼吸 | 卡頓、強制、機械 |

#### 4.2 轉化預示

當魔法少女接近墮落為屍骸時，其視覺元素會出現以下預示性變化：

| 元素 | 預示性變化 |
|-----|-----------|
| **光環** | 開始出現金屬質感的邊緣 |
| **枷鎖紋** | 紋理開始凸起，接近外露 |
| **變身鏡** | 鏡面開始生鏽或長出齒輪紋理 |
| **心匣** | 心跳聲混入齒輪轉動的聲音 |

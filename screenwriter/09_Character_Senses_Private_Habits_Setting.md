## **角色感官與私密習慣設定集**

本文檔專門記錄角色在遠離戰場、獨處時的**「非正常行為」**與感官尋求機制。這些設定作為「導演手冊」的延伸，為具體劇本或遊戲過場動畫提供視覺與動作參考。

### **主角 (晴香 Haruka) - 「控制」與「痛覺」的私密儀式**

#### **初期開朗時的私密習慣：「控制感」與「純潔」的追求**

在故事初期，晴香展現出的極度燦爛形象後隱藏著對失控的深層恐懼。她透過一系列私密、規律的行為習慣來維持這份脆弱的「內心平衡」。

- **規律的節奏感**：

    - **行為設定**：她在走路或獨處時，手腳會下意識地跟著某種內在的節奏輕微律動。她對日常的「節拍器」充滿執著——固定的上學時間、固定的午餐內容、固定的放學路線。這種規律性甚至延伸到她的呼吸與心跳。

    - **心理分析**：節奏對她而言是**「理性的節拍器」**。在一個充滿潛在爆發危機的內心世界中，她透過讓外界世界充滿音樂與規律，來確保那個混亂的、隨時會爆發的「增幅器」不被點燃。節奏 = 控制 = 安全。只要她能維持住這個節奏，她就能假裝自己是正常的、是可控的。

    - **象徵意義**：這暗示了她潛意識對「失控」的驚人恐懼。每當這個節奏被打亂（例如遇到彩），她的防線就開始瓦解。

    - **視覺參考**：
        - 獨處時輕拍大腿、輕聲哼唱固定的旋律
        - 走路時步伐異常規律，彷彿在跟著無人能聽見的鼓聲
        - 進食、整理房間時都遵循著某種儀式性的順序

- **對透明感的執著**：

    - **行為設定**：她非常喜歡透明的東西——清澈的玻璃杯、肥皂泡在陽光下的七彩折射、清晨露水的晶瑩。她會花很長時間靜靜地看著陽光穿透這些透明物質，觀察光線的折射與色彩的變化。她甚至會故意選擇透明的文具和衣著配飾。

    - **心理分析**：透明象徵著**「沒有秘密」與「純粹」**。因為她潛意識裡極度恐懼自己內心深處那個「殺死母親」的黑暗秘密，她下意識地追求視覺上的通透，試圖以此說服自己——「我是乾淨的、我是無害的、我內心裡沒有黑暗的東西」。透明物質的破裂聲音也讓她感到某種滿足感，彷彿這代表了什麼被「淨化」了。

    - **關鍵細節**：當她發現一個透明物體有裂痕或污漬時，她會感到一種莫名的不安——這無意識地反射著她對自我不完美的恐懼。

    - **視覺參考**：
        - 用手指尖輕輕觸碰透明物體的表面，檢查是否有任何瑕疵
        - 長時間凝視窗外的雨滴或清晨的露水
        - 集中收集各種透明的飾品與文具

- **微小的成癮：極甜的熱可可**：

    - **行為設定**：每天睡前必須喝一杯濃稠到近乎病態的熱可可。她對糖分的用量異常挑剔，一定要甜到某個特定的程度才能讓她感到滿足。她甚至會拒絕喝「普通甜度」的飲料，寧可自己加糖。

    - **心理分析**：高糖分帶來的短暫多巴胺激增是她的**「安全閥」**。糖分能強行製造那種虛假的幸福感，暫時壓制內心的不安。這是她用來維持「永遠充滿活力」假象的化學手段。每一次喝下那杯甜得不能再甜的熱可可，就像是在告訴自己：「我可以製造幸福，即使那個幸福是虛假的。」

    - **象徵意義**：這也反映了她對「正面情緒」的病態渴望。她不是在品嚐可可的味道，而是在「吞嚥幸福的假象」。

    - **視覺參考**：
        - 精心調配可可粉與糖的比例，動作緩慢而專注
        - 喝時閉上眼睛，彷彿進行某種儀式
        - 杯子見底後，會用指尖刮取杯底的糖分殘留物

#### **面對難關時的感官尋求：「痛覺確認」與「聲音的喪失」**

當晴香開始失去味覺並經歷情感耗損時，她的行為會從「維持平衡」轉向「尋找實感」。

- **冰冷的麻木：吃冰塊**：

    - **行為設定**：隨著味覺逐漸喪失，她不再依賴熱可可的甜味刺激。取而代之的是，她開始在沒人看見時，像嚼餅乾一樣嚼碎堅硬的冰塊。她尤其喜歡在冰塊完全冷凍、堅硬到最大程度時進行這個行為。

    - **心理分析**：當可樂失去甜味，只有牙齦的刺痛感與寒冷能穿透那層厚厚的情感麻木。這與美夜子游泳的思路一致——既然感受不到「愛」，就透過「痛」來確認自己還沒徹底變成空殼。她需要物理性的痛覺來證明自己還活著。

    - **象徵意義**：冰塊的破碎代表著她防線的崩潰。每一次咀嚼冰塊，都是她在自我傷害與自我確認之間的掙扎。

    - **視覺參考**：
        - 在廚房製冰盒前停留，用舌尖試探冰的溫度
        - 咀嚼時表情平靜無波，彷彿感受不到痛楚
        - 偶爾會有冰水從嘴角滑落，但她不去擦拭

- **無聲的共鳴：斷弦的吉他**：

    - **行為設定**：她的吉他斷了一根弦，但她拒絕修理它。相反地，她會在深夜用力撥弄那根斷掉的、會割傷手指的金屬絲。她甚至有時會故意讓它割傷她，然後靜靜地看著血珠滑過琴身。

    - **心理分析**：斷弦代表她破碎的靈魂。手指被割破的痛楚能讓她短暫地「感覺」到音樂——即使那是痛苦的音樂。這是她對「音律少女」主題最悲哀的反抗。她曾經能用吉他傳達快樂與希望，但現在她只能用血與痛來「演奏」。

    - **關鍵細節**：在這個過程中，她從不發出聲音。無聲的出血、無聲的痛楚、無聲的絕望——這代表著她最終的孤獨。即使在自傷，她也要確保沒有人聽到、沒有人看到。這正是她「善良的絕緣體」本質的終極體現。

    - **視覺參考**：
        - 深夜時分坐在房間角落，用食指輕輕撥弄斷弦
        - 血珠在吉他棕色的木質表面形成小小的珍珠
        - 用吉他的聲音（或無聲）作為背景，靜靜流血

- **強迫性的遮蓋：膠布纏繞** ^膠布習慣

    - **行為設定**：晴香隨身攜帶大量膠布。她見血即貼，對流血有極度恐懼。她的手指經常纏繞著膠布（隱藏自傷痕跡或微小傷口），即使傷口已經癒合，她也會繼續貼著新的膠布，形成一種強迫性的儀式。

    - **心理分析**：這是她對[[導演/01_創作核心與指導原則#^膠布哲學|膠布哲學]]的個人體現。表面上她說「貼住就看不到痛」，但深層是她潛意識恐懼「流血 = 死亡 = 母親/美夜子」（Alpha線記憶殘留）。她用膠布欺騙自己「傷口不存在」，這與她用「永遠樂觀」的面具遮蓋內心黑暗的行為邏輯一致。

    - **矛盾點**：她教校霸用膠布「記住痛」，卻自己用膠布「逃避痛」。這份矛盾是她角色弧光的核心——從「遮蓋者」成長為「直視者」。

    - **劇情弧線**：
        - 初期：過度使用膠布（逃避）
        - 中期：膠布開始「滲漏」——血從膠布縫隙滲出（無法遮蓋）
        - 高潮：撕開膠布（面對），皮膚龜裂但她說「這份痛楚...才是我還活著的證據」
        - 結局：不再需要膠布（整合）

    - **視覺參考**：
        - 書包裡有一個專門放膠布的小袋子，裡面有各種圖案的膠布
        - 手指上經常有2-3層疊加的膠布，形成厚厚的「護甲」
        - 幫別人貼膠布時手很穩，但看到自己流血時會微微發抖

#### **IG使用習慣：公開偶像帳號 vs 匿名厭世小號** ^晴香IG

晴香作為偶像，有一個光鮮亮麗的公開帳號——專業的打光、精心策劃的文案、源源不絕的正能量。但她也有一個秘密的匿名小號，只有5個粉絲（全是機器人或陌生人）。

- **公開帳號**：
    - 頭像是官方宣傳照，每篇貼文都經過經紀人審核
    - 內容：舞台照、排練花絮、與粉絲的正能量互動
    - 她是這個小圈子的「KOL」，總是強行拉隊友們拍照

- **匿名小號**：
    - 頭像是路邊拍的野貓，ID是亂碼（貓貓好可愛_78234）
    - 內容：負能量、罵髒話、深夜的厭世獨白、或者只是拍路邊的野貓和天空
    - 在那裡她不是偶像，也不是魔法少女，只是個普通的厭世少女
    - 這是她的「[[#10_少女日常與社交互動設定#秘密基地與「無人知曉的自己」|秘密基地]]」——唯一不需要表演的空間

- **心理分析**：公開帳號是她「維多利亞天使」形象的延伸，匿名小號則是[[#^膠布習慣|膠布]]遮蓋不住的真實。當她社死後，公開帳號的評論區從「好可愛」變成黑粉謾罵，她再也沒有更新過。

- **視覺參考**：
    - 公開帳號：專業打光、精緻構圖、永遠微笑
    - 匿名小號：模糊的深夜街景、野貓的特寫、沒有人臉的照片
    - 她在公開帳號幫隊友用可愛貼圖遮住比中指的手

### **妹妹 (美夜子 Miyako) - 「活著」的絕望掙扎**

#### **感官尋求與自我確認機制：身體的「活著」實感**

美夜子對「生存實感」的渴望，透過一系列私密、極端的感官行為得以體現。這些行為並非單純的自傷傾向，而是她在「解離狀態中確認自我存在」的扭曲方式。

- **長泳與呼吸掙扎**：

    - **行為設定**：她會在無人的長泳池中進行長距離游泳，故意將自己推向體力極限，迫使身體進行劇烈的換氣。她甚至享受「快要溺水」的瀕死感。

    - **心理分析**：對她而言，呼吸的掙扎是**「存在性的確認」**。身為被強行復活的人造人，她的每一次呼吸都是「借來的」。在游泳時為了生存而產生的強烈換氣，能讓她那近乎死寂的肉體產生最強烈的**「活著的信號」**——只有當她感到必須用力才能呼吸時，她才真實地感受到自己還「完整地活著」。這也呼應了她對生命本質的扭曲理解：**「生命 = 與死亡的掙扎」**。

    - **象徵意義**：水的模糊感象徵著她對現實的「疏離感」。在水下，外界的目光無法到達，她能暫時從作為「兵器」或「責任」的身份中逃脫，進入一個只有她與生死掙扎的私密時空。

    - **視覺參考**：
        - 游泳動作越來越急促，最後變成近乎瘋狂的拼命掙扎
        - 在水中停留，讓自己慢慢下沉，然後在最後關頭衝出水面
        - 濕漉漉的頭髮貼在臉上，但她無神的眼睛看著遠方
        - 泳池邊緣的抓痕，是她反覆嘗試極限的證據

- **黑暗中沖涼與黑暗的擁抱**：

    - **行為設定**：她喜歡在完全關閉燈光的浴室中沖涼，長時間地站在黑暗裡，讓熱水沖過身體。她享受黑暗帶來的「感官剝奪」。

    - **心理分析**：黑暗是一種**「解離的物理化身」**。黑暗消除了「觀測」——沒有光，她就不會被看見，不會被評判，不會被定義。對她而言，黑暗不是恐懼，而是唯一的**「安全庇護所」**。

    - **象徵意義**：黑暗如同母體或死亡，提供了一種「回歸」的幻想。在黑暗中，她不需要是「完美的兵器」或「倖存者」，她可以暫時放下一切防線，感受一種近乎於被溫柔包裹的感覺——彷彿有什麼無形的力量在擁抱著她受傷的靈魂。

    - **關鍵細節**：她在黑暗中會自言自語，輕聲呢喃一些人名——那些她無法拯救、已經消失的同伴名字。黑暗成為了她「悼念死者」的私密空間。

    - **視覺參考**：
        - 黑暗中的水流聲與她輕微的抽泣聲混在一起
        - 冷水忽然沖下時，她會瑟縮，彷彿被某種溫柔的懲罰
        - 她的手在黑暗中無序地撫摸著浴室的牆壁，尋找某種不存在的連結
        - 洗完後，她會在黑暗中靜站許久，不立即離開

- **生理性存在確認：最終的自我測試**：

    - **行為設定**：恢復人形態後，當長泳的窒息感與黑暗的感官剝奪都無法喚醒她的感受時，美夜子會在絕對隱密的環境下，透過自我刺激來進行最原始的「存在確認」。這不是為了快感，而是為了測試這具「借來的身體」是否還會對刺激產生反應——生理性的潮紅、加速的心跳、無法控制的顫抖、甚至滑落臉頰的生理性眼淚。只有在這個時刻，她那張永遠冰冷的臉，才會短暫地出現「像活人一樣」的表情。

    - **階段限定**：這個行為僅存在於她「恢復人形態後、尚未解決核心心理創傷」的過渡期。當她最終能夠在晴香面前「大哭」——獲得真正的情感釋放與被接納——這個生理性的「存在測試」將失去意義，自然消失。

    - **心理分析**：這是她「解離狀態中確認自我存在」的最後防線。性反應是人類極少數能繞過理智壓抑、直接引發自律神經系統反應的生理機制。對於情感迴路已經壞掉的美夜子而言，這是唯一能穿透那層厚重麻木的方式。她不是在追求快感，而是在進行**「感覺神經的通電測試」**——如果這具身體還會有反應，就證明她還沒有徹底變成空殼。

    - **與「大哭渴望」的殘酷對照**：這個行為最悲涼之處，在於它與她「終極渴望」的對比。她能透過生理刺激製造「眼淚」，但那不是真正的哭泣；她能讓臉頰泛紅，但那不是因為被愛而臉紅。她渴望的是在某人面前「大哭一場」——那是**「被無條件接納」**的眼淚。而獨自進行的生理性反應，只是**「確認自己還有肉體」**的證據。前者是靈魂的釋放，後者是對靈魂缺失的絕望確認。

    - **象徵意義**：這是「肉體在燃燒，靈魂在結冰」的終極體現。她的身體還會對刺激產生反應，但她的心已經感受不到任何東西。生理性的高潮與情感性的連結之間的鴻溝，正是她「被剝奪死亡權利」後最深層的創傷——她被迫活著，但她的靈魂早已冰封。

    - **視覺參考**：
        - 這個場景不應被直接呈現，而是透過暗示性的「之後」畫面來傳達
        - 她獨自躺在黑暗中，眼角有乾涸的淚痕，但眼神依然是空洞的
        - 她的手指會無意識地輕微顫抖，彷彿身體還記得那短暫的「活著」的感覺
        - 她會在這之後更長時間地凝視自己的手，彷彿在確認「這真的是我的手嗎」
        - 她的表情會從生理性的餘韻，迅速恢復成那張「死人臉」——這個轉變本身就是最大的悲劇

#### **強迫性的修復：研磨無法修復的斷刃** ^研磨斷刃

美夜子對凜的死有著無法消解的愧疚。這份愧疚具象化為一把斷刃匕首，以及對它進行病態保養的儀式。

- **信物設定：刻名匕首 (The Inscribed Dagger)**

    - **外觀**：一把與法師優雅形象格格不入的、實用主義的軍用匕首。刀柄纏著美夜子常用的繃帶（作為防滑），刀身刻有凜的名字。刀刃已經斷了一截，上面有洗不掉的銹跡（乾涸的血）。

### **反派 (黑奏/彩) - 破碎靈魂的儀式**

#### **膠布哲學的體現：修正/抹除 = 直接切除神經** ^黑奏膠布

黑奏的「[[導演/01_創作核心與指導原則#^膠布哲學|膠布]]」是最極端的形式——她不遮蓋傷口，不麻醉傷口，而是**直接切除感受痛苦的能力**。

- **對他人的「治療」**：她用這種方式「幫助」他人——抹除痛苦的記憶，切斷與傷口的連結。在她看來，這是仁慈；在故事的視角看來，這是剝奪了「態度自由」的暴行。
- **諷刺的自我矛盾**：她能切除他人的痛覺，卻無法對自己下手。螢的死是她靈魂深處那道永遠沒有被治癒的傷口——她所有的「修正」行為，其實都是在逃避這道無法面對的傷疤。
- **與其他角色的對比**：
    - 晴香用膠布「遮蓋」傷口（不看就不存在）
    - 朱音用戰鬥「麻醉」傷口（感覺不到就不痛）
    - 操用妝容與變身「偽裝」傷口（假裝不存在）
    - **黑奏用修正「切除」傷口**（沒有神經就不會痛）——這是膠布哲學的極端終點
- **主題意義**：黑奏代表了「接納 vs 修正」衝突中「修正派」的極致。她無法接受痛苦是存在的一部分，所以試圖從根本上消滅痛苦的感知能力。但情緒守恆定律告訴我們，痛苦不會消失，只會累積——最終以更可怕的形式反噬。

### **屍骸獵人 (朔) - 感官與身份**

- **心理分析**：一種幻想性儀式，將物理現象（金屬溫度）賦予超自然意義（靈魂共鳴）。他用耳環的溫度作為"她還記得我"的證據，當真相落空時，用肉體疼痛取代精神絕望。

### **賞金獵人 (朱音) - 失敗與母性的循環**

#### **膠布哲學的體現：戰鬥的腎上腺素 = 止痛藥** ^朱音膠布

朱音的「[[導演/01_創作核心與指導原則#^膠布哲學|膠布]]」不是遮蓋傷口，而是**麻醉**傷口。她用戰鬥中腎上腺素激增帶來的「High」狀態，來壓制內心對弟弟之死的痛苦。這與藥物成癮的機制完全相同——不是治癒，只是暫時感覺不到痛。

- **矛盾點**：止痛藥退效時，痛得更厲害。她越依賴戰鬥來「麻醉」，對戰鬥的需求就越大，最終形成惡性循環。
- **與其他角色的對比**：
    - 晴香用膠布「遮蓋」傷口（不看就不存在）
    - 美夜子用研磨「修復」傷口（物理行動代替情緒處理）
    - 朱音用戰鬥「麻醉」傷口（感覺不到就不痛）
- **關鍵場景暗示**：當朱音撕開晴香幫她貼的膠布時，她說的那句「妳以為貼住張貼紙，個傷口就會好？」——其實是她對自己的質問。她知道麻醉不是解藥，但她無法停止。

#### **膠布哲學的補充：粗糙繃帶的實用主義** ^朱音繃帶

朱音對待**肉體傷口**的態度是純粹的「實用主義」——她只用最粗糙、最廉價的繃帶，隨便纏幾圈就算了。不講究消毒、不在乎美觀、不管會不會留疤。唯一標準：「止住血，能繼續戰鬥就好。」

**與晴香的強烈對比**：

| 晴香 | 朱音 |
|-----|-----|
| 隨身攜帶各種圖案的可愛膠布 | 只用最便宜的醫療繃帶 |
| 見血即貼，動作細心 | 傷口流血也懶得處理，除非影響戰鬥 |
| 膠布 = 遮蓋痛苦的儀式 | 繃帶 = 維持戰鬥機能的工具 |
| 對傷口有恐懼（死亡聯想） | 對傷口麻木（已經習慣了） |

- **心理分析**：這不是「不值得被好好對待」（那是美夜子的邏輯），而是「傷口根本不重要」——對於一個戰鬥成癮者來說，肉體只是武器的載體。她的身體已經遍布舊傷疤，再多一道也無所謂。這與她用戰鬥「麻醉」心理傷口的邏輯一致：物理傷口和心理傷口一樣，「不去感覺就不存在」。
- **視覺參考**：
    - 她的手臂和腿上經常纏著鬆垮的、髒污的繃帶
    - 繃帶下有時會滲出血跡，但她渾然不覺
    - 她從不買新繃帶，經常重複使用舊的（病態囤積習慣的延伸）

#### **習慣0: 穿鞋守門 (Sleeping in Boots at the Threshold)** ^穿鞋守門

- **行為設定**：朱音從不睡在床上。她永遠睡在房間入口附近——門口的地板、玄關的角落。她睡覺時永遠穿著軍靴（便於逃跑/戰鬥）。她的房間有一張整潔的床，但那是給「正常人」用的——她自己從未躺上去過。她會在門口堆放一些小物件（空罐、碎玻璃），作為「入侵者警報」。由於姿勢不舒服，她無法自然入睡，只能依賴情緒毒品強制關閉意識。

- **心理分析**：這是[[02_次要角色背景故事#^block-akane|兩段打手經歷]]留下的極端警戒狀態的具象化。她的潛意識永遠在等待「那一刻」——門被踢開、敵人衝進來。
    - 穿著軍靴睡覺 = 「我隨時準備好戰鬥或逃跑」
    - 睡在門口 = 「我是第一道防線，也是唯一的防線」
    - 從不用床 = 她認為自己不配擁有「安全感」這種奢侈品
    - 依賴毒品入睡 = 她的身體拒絕放鬆，只有化學強制才能讓她的警戒系統暫時關閉

- **惡性循環**：不舒服的睡姿 → 需要毒品才能入睡 → 毒品依賴加深 → 對戰鬥的需求增加（換取毒品）→ 更多罪行 → 更深的恐懼 → 更極端的警戒

- **象徵意義**：這個習慣與她的「病態囤積」形成呼應——門口堆放的空罐頭既是「入侵者警報」，也是她囤積習慣的另一用途。她把自己變成了「永不關閉的哨兵」，卻也因此永遠無法真正休息。

- **視覺參考**：
    - 房間裡整潔的床，床單從未被弄皺
    - 門口的地板上有一條薄毯子和一個扁平的枕頭
    - 穿著軍靴蜷縮在門邊的朱音，手邊放著武器
    - 門前堆放的空罐頭形成簡易警報線
    - 她的雙腳因長期穿著軍靴而磨出水泡和繭

#### **關鍵場景：脫鞋的權利 (The Right to Remove Your Boots)**

- **場景設定**：深夜。朱音的臨時住所/宿舍。晴香因某事需要找朱音，發現門沒鎖（朱音藥效昏睡中）。

- **發展**：
    1. 晴香推門進入，看到昏暗房間裡整潔的床——空的
    2. 她順著微弱的呼吸聲，在門口角落發現蜷縮的朱音
    3. 朱音穿著軍靴，姿勢扭曲，手邊是空的藥瓶和一把匕首
    4. 晴香注意到朱音的軍靴繫帶勒得很緊，腳踝有明顯的勒痕和磨傷
    5. 晴香輕輕蹲下，開始解開軍靴的繫帶

- **關鍵動作**：
    - 晴香脫下朱音的軍靴，看到裡面的腳——水泡、老繭、還有幾處已經磨破結痂的傷口

    7. 晴香：「所以，妳對腳可以透下氣啦。」
    8. 朱音低頭看著那些印著小貓、小花的膠布，沉默良久

- **台詞的意義**：
    - 「我幫妳睇住度門」= 我來承擔妳的恐懼
    - 「尋晚無人拍門，今晚都唔會有」= 用事實（而非空洞安慰）告訴她：妳是安全的
    - 「妳對腳可以透下氣啦」= 妳可以卸下武裝了，至少在這一刻

- **象徵意義**：
    - 脫下軍靴 = 卸除「隨時準備戰鬥/逃跑」的武裝狀態

    - 朱音想撕掉膠布但沒有 = 她第一次接受被「好好對待」
    - 晴香坐在門口守夜 = 用行動（而非言語）建立信任
    - 朱音第一次「被保護」而非「保護他人」= 她母性投射的對象（晴香）反過來照顧了她

#### **習慣1: 長期咀嚼同一塊香口膠**

- **行為設定**：朱音有一塊她咀嚼了好幾個月的香口膠，放在口腔裡不斷反覆咀嚼。不同的香口膠款式，但咀嚼時間都很長。這塊膠已經變色、變形，失去了原來的香氣，但她依然每天咀嚼。

- **心理分析**：源自弟弟誤食情緒結晶而死的創傷，她無法放棄任何食物相關的物品。長期咀嚼同一塊膠，象徵著無法"吞嚥"的痛苦——好像只要還在咀嚼，就能證明"我沒有浪費，我沒有失敗，我沒有害死他"。

- **象徵意義**：香口膠 = 被循環咀嚼的痛苦；不捨得吞下 = 對失敗的無法釋懷。

- **視覺參考**：面無表情地不斷咀嚼，下頜機械運動。口腔裡那塊褪色的膠，早已沒有味道。

#### **習慣2: 養死植物的循環**

- **行為設定**：朱音強迫性地買植物、種植、澆水——澆得太多、照顧過度，最終總是弄死它們。植物死後她會崩潰、哭泣，但過幾天又重新開始買新植物、種植、澆水、弄死。無盡的循環。

- **心理分析**：對弟弟死亡的強迫性重複。她無法停止嘗試"拯救"，因為停止等於承認自己無力。每一次植物的死亡都是弟弟的死亡再現，每一次購買新植物都是"這次我一定能救活它"的妄想。

- **象徵意義**：植物 = 被重複犧牲的弟弟；澆水過度 = 無法正確愛護；死亡循環 = 無力感與絕望。

#### **習慣3: 餵貓形態的美夜子**

- **行為設定**：朱音成日拎著食物去餵處於貓形態的美夜子。她會說："乖貓貓，吃飯囉。"用母親的語氣照顧美夜子，彷彿美夜子是她"養活成功的孩子"。這是對弟弟死的某種扭曲補償。

- **心理分析**：無法拯救弟弟，所以將母性投射到美夜子身上。照顧美夜子讓她感覺"至少有一次我成功地養活了某個生命"，緩解了"我無法救活弟弟"的無力感。

- **象徵意義**：美夜子 = 被"重新母性化"的弟弟；餵食 = 對失敗的補償。

- **視覺參考**：溫柔地拎著食物，蹲下對著貓形態美夜子。用母親語氣說話，眼神充滿母性關懷。

- **隱藏的回憶觸發**：雖然美夜子失去了人形態的記憶碎片，但在貓形態中，朱音的「溫暖母性語氣」有時會在美夜子的無意識深處引發片段記憶閃現——那些關於凜的、關於「被照顧」的感受。美夜子的耳朵會抖動，眼神會短暫變得清明而悲傷，彷彿在回應某個已經死去的聲音。朱音察覺不到，但這些微妙的身體反應，暗示著美夜子的靈魂仍在某處記得那些失去的人。

#### **習慣4: 餵養野狗 (Feeding the Strays)** ^餵養野狗

- **行為設定**：朱音會在深夜的暗巷餵養流浪狗——不是可愛的小狗，而是那種髒兮兮、會咬人、被所有人驅趕的癩皮狗。她餵的食物往往是她囤積的過期罐頭，或是她自己捨不得吃的食物。她會蹲在垃圾堆旁邊，看著野狗狼吞虎嚥，眼神複雜。

- **心理分析**：她在野狗眼裡看到的不是「可憐」，而是「飢餓」。當年她為了不讓弟弟餓肚子才去混黑道，結果卻因為飢餓害死了他（弟弟因飢餓誤食她私藏的毒品）。餵狗是一種病態的補償——看著野狗吃飽，她能獲得「這次我餵飽了他」的虛假安慰。她餵狗的食物，或許就是她當年想買給弟弟吃的那些普通食物。野狗狼吞虎嚥的姿態，和弟弟當年吞下「糖果」的畫面重疊。

- **關鍵台詞設計**：當有人叫她不要靠近瘋狗時：
  「牠們不是壞，牠們只是餓。餓的時候，誰都會變成怪物的……就像我細佬一樣。」

- **與[[#^屍骸不倫|屍骸不倫關係]]的連結**：餵野狗的行為是她後來餵養魔法屍骸的原型。野狗、魔法屍骸、朱音——這三者都是「飢餓者」與「被社會遺棄者」。透過朱音的視角，故事將挑戰「人類」與「怪物」的界線，暗示[[03_世界觀設定#^trinity-spectrum|三位一體光譜]]的深層含義。

- **視覺參考**：
    - 陰暗潮濕的暗巷，垃圾堆旁邊
    - 癩皮狗警戒地盯著她，但聞到食物後慢慢靠近
    - 朱音蹲著，手上拿著過期罐頭，眼神複雜（不是憐憫，是某種認同）
    - 狗吃完後用舌頭舔她的手，她沒有躲開
    - 視覺對位：朱音柔軟溫熱的人類肌膚 vs 野狗粗糙的舌頭，預示著後來與屍骸的親密接觸

#### **習慣5: 與無名屍骸的不倫連結 (The Forbidden Bond with the Nameless Corpse)** ^屍骸不倫

這是朱音最隱秘的行為——身為專門獵殺屍骸的魔法少女，她卻私下餵養一隻魔法屍骸，並與之發展出某種扭曲的親密關係。這段關係是對社會規則的徹底背叛，也是她對「正常世界」的無聲抗議。

- **相遇設定**：朱音在某次任務後的暗巷發現一隻「異常」的魔法屍骸。這隻屍骸沒有主動攻擊，而是在翻垃圾、啃食魔力殘渣——姿態和她餵養的野狗一模一樣。那雙眼睛——充滿戒備、飢餓、孤獨——讓她無法下手。

- **關係發展**：
    1. **初期：餵養**
       - 朱音開始偷偷「餵養」這隻屍骸（用自己的魔力殘渣或低純度的情緒結晶）
       - 這直接呼應了她的創傷——她再次為了「養活」某個怪物而觸犯禁忌

    2. **中期：習慣化**
       - 屍骸開始認得她的氣味，不再對她有敵意
       - 朱音發現自己會期待這些「餵食時間」——這是她唯一不需要戰鬥、不需要麻醉自己的時刻

    3. **深化：肉體接觸**
       - 某次朱音受傷或被「正常人類社會」迫害後，躲在暗處
       - 屍骸出現了，用那雙非人的、冰冷的手笨拙地觸碰她的傷口
       - 朱音沒有躲開。她讓屍骸擁抱她——那種冰冷異質的觸感，反而讓她感到某種扭曲的安全感

- **「不倫」的本質**：
    - 身為專門獵殺屍骸的魔法少女，她卻私下餵養屍骸、與屍骸有肉體接觸
    - 這是對社會規則的徹底背叛，也是她對「正常世界」的無聲抗議
    - 在屍骸面前，她不需要是「英雄」或「戰士」，她可以只是一個「充滿罪惡感的姊姊」
    - 屍骸不會批判她害死弟弟，只會依賴她給予的生存資源——這是她唯一能獲得的「無條件接納」

- **親密行為的心理分析**：
    - 朱音在屍骸身上投射了「死去的弟弟」——她無法拯救弟弟，所以試圖「飼養/拯救」這隻屍骸
    - 這是一種病態的補償心理，也是她對「飢餓者」的深層共情
    - 屍骸的觸感（冰冷、非人）反而讓她感到安心——因為「正常人類」的溫暖只會讓她想起她無法保護的弟弟

- **主題服務：三位一體的暗位**：透過這段關係，觀眾被迫思考：
    - 細佬（人類）為了毒品可以變得像鬼一樣貪婪
    - 野狗（獸）為了護食會咬人，但給予溫暖就會忠誠
    - 屍骸（魔）殺人是因為生存本能，但對朱音展現的保護欲，卻比任何「正常人類」都更純粹
    - **那麼，「人」和「怪物」的分別到底在哪裡？**
    - 這段關係呼應了[[03_世界觀設定#^trinity-spectrum|三位一體光譜]]的核心概念——人類、魔法少女、魔法屍骸只是同一條光譜上的不同位置

- **結局伏筆：從餵養者到首領**：
    - 透過長期的「餵養」與「共處」，朱音與這隻屍骸建立了深層的精神連結
    - 這種連結不是控制，而是「共鳴」——她理解屍骸的飢餓，屍骸理解她的孤獨
    - 當朱音最終與屍骸群體建立連結時，這隻「無名屍骸」成為她身邊的特殊存在
    - 不是僕從，而是「伴侶」或「右手」——唯一一個從她還是「人類」時就陪伴她的存在
    - 這段關係的「真實性」，正是故事想要傳達的核心——即使是人與怪物，也能建立真實的連結

- **關鍵場景設計**：

  **場景 A：第一次餵食**
    - 地點：任務後的廢棄工廠
    - 朱音發現屍骸在翻垃圾，姿態如野狗
    - 她沒有攻擊，而是扔出一塊低純度的情緒結晶
    - 屍骸戒備地看著她，然後撲向結晶狼吞虎嚥
    - 朱音：（獨白）「……和牠們一樣。只是餓了。」

  **場景 B：受傷後的擁抱**
    - 地點：陰暗的巷弄
    - 朱音被「正常人類」（可能是黑道追殺或社會排斥）傷害後，躲在角落
    - 屍骸出現，用非人的手觸碰她的傷口
    - 朱音沒有躲開，反而靠向那冰冷的懷抱
    - 她輕聲說：「沒關係……你和佢（細佬/野狗）都一樣，只是想活下去而已。我都係。」

- **視覺參考**：
    - 朱音坐在陰暗的巷弄裡，一邊是髒兮兮的野狗趴在她膝蓋上，另一邊是恐怖的魔法屍骸蜷縮在她身邊
    - 她同時撫摸著這兩個「怪物」——這個畫面直接質問：「人」和「怪物」的界線在哪裡？
    - 朱音柔軟溫熱的人類肌膚 vs 屍骸堅硬異質的外殼形成強烈對比
    - 屍骸笨拙地用爪子撫摸朱音的頭髮（模仿她撫摸野狗的動作）
    - 兩人/一人一獸蜷縮在陰暗角落，姿態如同野狗依偎

#### **IG使用習慣：深夜Emo、食物遺照、野狗醜照** ^朱音IG

朱音一開始根本沒有帳號。她覺得社交媒體是「有錢人的玩意」，直到被晴香磨了很久才勉強開了一個。

- **帳號狀態**：
    - 頭像是她餵養的那隻野狗（[[#^餵養野狗|癩皮狗]]的醜照）

- **心理分析**：她的IG風格反映了她「實用主義」的世界觀——她不需要被「看見」，她只是在記錄。但那些深夜的孤獨街景、偷拍隊友的照片，其實是她與世界連結的微弱嘗試。

- **IG敘事線中的角色**：

### **木偶師 (操) - 身份剝奪與清潔**

#### **習慣1: 戴眼罩避開鏡子**

- **行為設定**：操喜歡在房間套上眼罩。她的房間被父親堆滿了鏡子——因為父親知道她不喜歡照鏡子，所以故意這樣做，強迫她"面對自己"。眼罩讓她避開父親的暴力善意，在黑暗中擁有一點點自主權。

- **心理分析**：被父親篡改身體後，她無法面對鏡中的自己——那個不是她選擇的身體。眼罩是對父親控制的逃避，同時也是對身體認同障礙的自我保護。

- **象徵意義**：眼罩 = 對鏡像的拒絕；黑暗 = 逃脫父親的控制；鏡子 = 父親的暴力善意。

- **視覺參考**：戴著眼罩，在堆滿鏡子的房間裡摸索移動。偶爾摘下眼罩看到鏡中自己時，眼神閃過痛苦。

#### **習慣2: 強迫性清洗儀式**

- **行為設定**：操每天長時間擦洗身體，用滾燙熱水沖淋，直到皮膚擦破、滲血。彷彿要把什麼"洗掉"——父親的觸碰、被篡改的身體、性別認同的混亂。她會反覆清洗同一個身體部位，直到皮膚潰爛。

- **心理分析**：典型的創傷後清潔強迫症。她在用肉體的傷害來象徵性地"清潔"無法被洗掉的精神污漬——父親對她的傷害、身體的背叛感。

- **象徵意義**：清洗 = 對污漬的永恆無望追求；滾燙熱水 = 自我懲罰；擦破皮膚 = 徹底的身體厭惡。

- **視覺參考**：在浴室長時間沖淋，水蒸氣瀰漫。用力擦洗身體，皮膚發紅、破裂、滲血。表情痛苦但執著，彷彿無法停止。

#### **習慣3: 聲音控制儀式**

- **行為設定**：操說話永遠輕如耳語，音量從不超過對話所需的最低限度。如果她不得不提高音量（如緊急情況），事後她會陷入數小時的焦慮，反覆回想自己是否「說得太大聲」或「表現得太失態」。

- **心理分析**：這不是害羞，而是**「低音量代表不會犯錯」**的強迫性信念。聲音是最容易「暴露破綻」的媒介——說錯話、語氣不對、聲音不夠優雅，都可能讓人「看穿」她的表演。

- **象徵意義**：耳語 = 減少犯錯的可能性；音量控制 = 對「正確形式」的極致追求；事後焦慮 = 對「被發現」的恐懼。

#### **習慣4: 物品擺放強迫**

- **行為設定**：她房間的所有物品都遵循精確的角度和位置。書本按高度排列，誤差不超過0.5公分；桌面物品與桌緣保持固定距離；衣櫥裡的衣服按顏色深淺排序。如果有人移動了任何東西，她會立刻發現並花大量時間「修復」，同時焦慮地想著「是誰碰了我的東西」。

- **心理分析**：她控制不了自己的身體和身份，所以瘋狂地控制外在環境。物品的秩序是她對混亂內心的補償——只要外在世界是「正確」的，她就能假裝自己也是「正確」的。

- **象徵意義**：精確擺放 = 對控制感的病態渴望；修復行為 = 對「被侵犯」的焦慮反應；固定角度 = 用數學來取代無法定義的「自我」。

#### **習慣5: 貓咪玩偶依賴**

- **行為設定**：她永遠帶著一隻黑貓玩偶（與美夜子的貓形態相似但她絕不承認）。私下獨處時，她會抱著玩偶發呆數小時，什麼都不做，只是感受「不需要表演」的片刻解脫。

- **心理分析**：玩偶是她唯一能「不演」的對象——玩偶不會評判她，不需要她維持完美形象。諷刺的是，她鄙視美夜子的貓形態（不符合「正確形式」），卻對貓咪玩偶有著病態的依戀。這暴露了她內心深處其實渴望「不需要符合形式也能被接受」的存在方式。

- **象徵意義**：黑貓玩偶 = 對美夜子的潛意識認同；抱著發呆 = 唯一不需要表演的時刻；絕不承認相似 = 對自身矛盾的否認。

#### **習慣6: 鏡子凝視儀式**

- **行為設定**：大部分時間她戴眼罩避開鏡子（習慣1），但偶爾會在深夜獨自站在鏡子前，長時間凝視自己的倒影，試圖找到「真正的我」。這個儀式從不會有結果——她看到的永遠只是一個「表演中的角色」，而不是真實的自己。

- **心理分析**：這是對習慣1的矛盾補充。逃避鏡子是因為害怕看到「被篡改的身體」，但偶爾的凝視是因為她仍然渴望在鏡中找到「真正的自己」。這種矛盾的行為模式反映了她身份認同的分裂狀態。

- **象徵意義**：深夜凝視 = 對自我認同的絕望追尋；找不到結果 = 身份被徹底篡改的悲劇；表演中的角色 = 她唯一能看到的「自己」。

#### **IG使用習慣：精緻生活濾鏡與秒讚但不敢留言** ^操IG

操的IG是她「完美形式」追求的延伸——一個精心策劃的「理想女性生活」展示。

- **帳號狀態**：
    - 一開始是小號，沒有頭像，只看不發
    - 秒讚晴香每一條Post但不敢留言
    - 會截圖保存每一張有自己的合照，放大檢查自己夠不夠「女孩子」

- **發文風格**：
    - 開始經營後，成為「精緻生活」帳號
    - 只有甜品、蕾絲、風景，濾鏡很重
    - 從不露臉，只有局部（精心塗好指甲的手、穿著蕾絲的手腕）

### **引路女孩 (愛莉) - 無聲的吶喊**

#### **習慣1: 用廢棄零件搭建實驗裝置**

- **行為設定**：愛莉用紙箱、膠帶、廢棄零件搭建各種"實驗裝置"。她希望有一天能拿給母親看，證明"我也可以像你一樣厲害"。但這些裝置往往簡陋無用，這讓她既感到自豪又感到絕望。

- **心理分析**：渴望被母親看見、認可。通過模仿母親的實驗行為，她試圖建立"我也屬於這個世界"的連結。但實驗的失敗預示了她最終無法被母親真正接納的命運。

- **象徵意義**：實驗裝置 = 求被看見的訊號；廢棄零件 = 破碎的希望。

- **視覺參考**：認真地膠帶貼零件，眼神充滿期待。完成簡陋裝置，既自豪又失望。看著空蕩蕩的房門，等待永遠不會出現的母親。

#### **習慣2: 寫給母親的無聲信件**

- **行為設定**：愛莉日復一日地寫信給母親：媽媽，今天我做了一個實驗…媽媽，你什麼時候來看我？每一封信都被仔細摺疊、保存。母親從未回信，但愛莉從不停止寫。

- **心理分析**：每一封信都是無聲的吶喊——"看看我、愛我、承認我的存在"。寫信本身成為了證明自己還活著的方式。當她最終變成石像時，所有未讀的信件成了她存在的唯一記錄。

- **象徵意義**：信件 = 單向的愛與求救；無回應 = 被母親忽視的命運；被保留 = 對失敗的記錄。

- **視覺參考**：低眉順眼地寫信，筆跡稚嫩而認真。信紙堆滿房間的一角。偶爾讀著自己的舊信，眼神空洞。

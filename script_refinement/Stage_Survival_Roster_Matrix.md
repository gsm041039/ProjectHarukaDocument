# 階段與存活名單矩陣

**文件目的**：追蹤所有主要角色在四幕結構下的狀態變化，解決「邊個出場、邊個已死」的混亂問題。

**追蹤角色**：
1. 晴香（雨宮晴香）
2. 美夜子（水無月美夜子）
3. 朱音（岩倉朱音）
4. 操（綾小路操）
5. 彩/黑奏（澄川彩，雙重人格）
6. 凜（綾瀨凜）

## 第一幕（起）：內在的戰爭

**時間範圍**：帝國歷 113年3月 - 114年3月

**核心目標**：
- 建立魔法少女團隊（招募朱音、操）
- 晴香與夕的對峙和融合
- 社會風評系統、少女日常描寫（潛入學校偷東西、假日約會）

**關鍵事件**：
- 彩以轉校生身份登場（113.3-113.6），晴香冷漠對待，只在校霸欺凌時幫過她，之後因事入隊
- 朔以屍骸獵人身份登場，阻撓晴香等人行動
- 美夜子（貓形態）的PTSD觸發，出現凜的零碎回憶畫面
- 夕的早期階段（相處期）：鏡中對話、毒舌軍師、晴香出現身體違和感
- 美夜子失蹤與救援（愛莉潛意識風暴）
- 晴香與夕和解：**部分融合**（第一幕末）——夕仍保留獨立意識，兩者共存

**特殊演出**：
- 晴香的"絕對共感視界"首次展示（觀眾以為是遊戲介面）
- 美夜子作為會說話的貓，限制視角（觀眾看不到內心）

## 第二幕（承）：世界的試煉

**時間範圍**：帝國歷 114年3月 - 5月初

**核心目標**：
- 對抗黑奏體制
- 恐怖家家酒14天事件
- 團隊成長但逐漸分裂

**關鍵事件**：
- 凜以**情緒管理局特工**身份登場（與朔一起），記憶被改寫，自己不知真相
- **觀眾在第二幕知道凜已死並被重組**（通過某種方式揭露）
- **美夜子身份揭露**：觀眾和晴香同時知道黑貓就是美夜子
- 恐怖家家酒14天開始（朱音缺席前13天，在下水道掙扎）
- 操精神崩潰邊緣（鏡像破碎之夜後）
- 凜多次與晴香接觸，被溫柔觸動
- 第二幕末：凜拒絕晴香治療，**逆向拆線**（心理儀式）

## 第三幕（轉）：偽高潮與崩塌

**時間範圍**：帝國歷 114年5月 - 6月

**核心目標**：
- 英雄失敗
- 陷入絕望
- 晴香被困二十年（體感）

**關鍵事件**：
- 恐怖家家酒第14天：嘆息之橋戰役（朱音變糖果山、操變人偶牆）
- **凜的自願改造**：拒絕晴香治療後，自願接受改造，變成 Unit 00 - Null
- **美夜子三段式覺醒**（嘆息之橋戰役後）：
  1. 貓殼碎裂 → 赤裸人類（**詛咒被美夜子的意志永久打破**）
  2. 月下窗台自慰測試（生理性流淚）
  3. 晴香擁抱 → 崩潰大哭
- **凜的最終戰與死亡**（嘆息之橋翌日清晨）：美夜子處決凜，履行血之約定
- 帝國廣場決戰：晴香第二次改變現實失敗，被困集體潛意識
- 二十年守望：夕在現實守護晴香肉身3-6個月（體感20年）

## 總結：角色命運軌跡

### 存活狀態
- **晴香**：存活 → 存活 → 被困 → **永恆守護者（唯一的神）**
- **美夜子**：貓形態 → 貓形態 → 人類形態 → **人類形態（唯一見證者/唯一連結）**
- **朱音**：存活 → 放逐（半屍骸狀態）→ **物理死亡（環境化：糖果山）** → **已故（精神永存/地標）**
- **操**：存活 → 精神崩潰邊緣 → **物理死亡（環境化：人偶牆）** → **已故（精神永存/地標）**
- **彩/黑奏**：分裂存在 → 分裂存在 → 分裂存在 → **犧牲/人格消散（永劫回歸）**
- **凜**：回憶/陰影 → 存活（敵對）→ **死亡（被處決）** → 死亡

### 關鍵轉變時刻
1. **第一幕 → 第二幕**：晴香與夕**部分融合**（夕仍保留獨立意識）
2. **第二幕 → 第三幕**：朱音放逐（半屍骸狀態）；操精神崩潰（鏡像破碎之夜）；凜改造；**美夜子身份揭露與覺醒**（嘆息之橋戰役，操朱音犧牲後貓殼碎裂變回人類）
3. **第三幕重大犧牲**：**朱音和操物理死亡**（環境化為糖果山和人偶牆）；凜被處決；晴香被困
4. **第三幕 → 第四幕**：時間重置；晴香甦醒；但**朱音和操的犧牲不可逆轉**
5. **第四幕結局**：彩覺醒犧牲；晴香成為永恆守護者

### 「黑暗治癒」核心：不可逆的犧牲

**重要概念說明**：

朱音和操在第三幕的死亡是**真正的、不可逆轉的物理死亡**。她們不是簡單的「屍骸化」，而是：

1. **由角色變成地標**：
   - 朱音 → 糖果山（發出甜膩香氣的巨大物理障礙）
   - 操 → 人偶牆（由絲線縫合的瓦礫與殘肢）

2. **第四幕的存在形式**：
   - **實體**：糖果山和人偶牆成為新世界的遺跡/紀念碑
   - **精神**：晴香作為永恆守護者可在精神層面感應到她們的殘存執念
   - **概念**：朱音代表「給予」，操代表「真實」，成為新世界規則的一部分
   - **絕對不是**：可以說話、行走、有自主意識的實體復活

3. **為何必須如此殘酷**：
   - 如果她們能在第四幕像普通人一樣復活，之前的「畢業」和「斷臂/暴食」犧牲就會變得廉價
   - **美夜子是唯一親眼見證她們犧牲的活人** - 這就是美夜子的詛咒
   - **晴香是唯一能與死者溝通的神** - 這就是晴香的孤獨
   - **朱音和操是這場戰爭中燃燒殆盡的薪柴** - 這就是黑暗治癒的代價

4. **悲劇美感的核心**：
   - 真正的犧牲不可逆轉
   - 倖存者背負著見證的痛苦
   - 死者以概念和遺跡的形式永存
   - 這種殘酷才是「黑暗治癒」的真諦
